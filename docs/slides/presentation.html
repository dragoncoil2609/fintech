Phần 2 · Hệ thống cốt lõi

The Reality Check.

Nhận đề bài AI Money Coach thì dễ. Nhưng làm thế nào để hệ thống chạy ổn định, chi phí rẻ, và dữ liệu trung thực trong đúng 48h thời gian đặt ra của cuộc thi?

Hạ tầng hệ thống của chúng tôi được thiết kế để chịu tải thật.

Kiến trúc tổng thể AWS

Bức tranh toàn cảnh: AWS VPC Architecture

Hệ thống được chia lớp rõ ràng (Edge, API, Compute, Data) trên Public/Private Subnets, thiết kế High Availability ngay từ đầu.

Hệ thống · Layer FE & Security

Layer FE & Security

CloudFront S3 cho Frontend tĩnh siêu nhanh.

ALB + WAF bảo vệ mọi traffic API.

Bài toán: Scale tĩnh cho Frontend & chống Bot. Giải pháp: Offload traffic tĩnh ra CloudFront, chốt chặn bảo mật bằng WAF trước khi chạm Backend.

Hệ thống · Layer BE

Layer BE (ECS Fargate + SQS)

  • Tại sao chọn ECS Fargate thay vì Lambda? AI Inference là bài toán chạy rất lâu (Long-running task). Fargate mang lại mức Dự báo chi phí (Cost Predictability) tối ưu nhất ở quy mô lớn.
  • Xử lý File sao kê khổng lồ: Kiến trúc Async. API đẩy việc vào SQS -> Fargate Worker âm thầm xử lý, không block UI.
  • Dữ liệu: RDS PostgreSQL (Trữ liệu an toàn) & ElastiCache Valkey (Rate limiting).
Hệ thống · Layer DB (Data Integrity)

Lặp dòng nạp vào = Tiền sai.

Tình huống thực tế: Sao kê tháng này chồng lên tháng trước -> Tiêu 1 triệu bị tính thành 2 triệu. User chửi app ngay lập tức.
Giải pháp Dedup 4 lớp khắt khe: VD: 2 giao dịch trùng [Ngày+Giờ+Số tiền] = 1. Xử lý triệt để cả lỗi Hoàn tiền (Refund) và Date tolerance.
Data Privacy Guarantee: File sao kê chỉ lưu trên S3 (Encrypted) và tự động xoá vĩnh viễn sau 1 giờ (Retention Policy).

Hạ tầng xịn đến đâu, nếu thuật toán sai thì Business vẫn sập. Chúng tôi scope độc lập theo user_id kết hợp sức mạnh Isolation của Cognito.

Bảo vệ integrity trước khi nhúng tay vào AI. Đây là nghiệp vụ ăn điểm tuyệt đối.
Hệ thống · Layer Monitoring

Observability: CloudWatch & SNS

Bài toán: Backend sập hay SQS kẹt Job làm sao hay?
Giải pháp: Setup CloudWatch Metrics theo dõi Error Rate & Queue Depth -> Bắn Alarm tự động qua SNS (Email/Slack). Biết lỗi trước khi khách báo.

Hybrid AI Pipeline: Parse OCR → Classify → Summarize.

→
2 · Bedrock Haiku 4.5
Chỉ dòng mơ hồ mới gọi LLM siêu rẻ.
→
3 · LocalAI fallback
Bedrock lỗi? Tự rớt về Local, hệ thống không bao giờ sập.

Truy vấn tự nhiên qua Text-to-SQL kèm Guardrail nghiêm ngặt chống SQL Injection. Tối ưu chi phí: Kết hợp Keyword (miễn phí) xử lý 65% dữ liệu, chỉ dùng Claude Haiku cho 35% còn lại.

Phần 3 · Kỹ thuật Prompting

Single-shot · Few-shot · JSON-only.

SYSTEM Vietnamese transaction categorizer → exactly one of: Food, Transport, Shopping, Bills, Entertainment, Health, … Respond ONLY JSON: [{"idx": int, "category": "<Category>", "confidence": "high|low"}] FEW-SHOT 1. HIGHLANDS COFFEE BUI VIEN | -65000 → Food / high 2. SALARY JUNE 2026 | 28000000 → Salary / high

Batch 10 dòng/lần gọi · System prompt + Few-shot tách riêng để tận dụng tính năng Prompt-Cache của Bedrock, tiết kiệm tối đa token.
